# WurmTools.Desktop

**WurmTools.Desktop** is an open-source, offline-first desktop toolkit for *Wurm Online* players.  
It focuses on performance, transparency, modular design, and experimental development.

This project is also an experiment:

> **How far can a real-world desktop tool go using AI-assisted development, combined with human domain knowledge and curation?**

---

## Project Philosophy

This project follows a **human-in-the-loop** approach.

### Human-driven
- Concept and overall direction
- Architectural decisions
- Dataset curation, validation, and structure
- Review, testing, and final acceptance

### AI-assisted
- Code generation and iteration
- Refactoring and optimization suggestions
- Boilerplate and repetitive implementation work

This is **not** a ‚Äúpure AI-generated project‚Äù, nor a one-click result.  
AI is used as a tool ‚Äî **not as an authority**.
